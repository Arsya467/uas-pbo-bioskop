# Deskripsi

Program ini merupakan aplikasi berbasis terminal (CLI) untuk mengelola data jadwal film bioskop. Program ini dibuat menggunakan konsep Pemrograman Berorientasi Objek pada bahasa Dart.

# Fitur

- Tambah Film
- Lihat Semua Film
- Cari Film
- Hitung Total Harga
- Simpan Data
- Keluar

# Konsep OOP yang Digunakan

- Class & Object
- Encapsulation
- Inheritance
- Polymorphism
- Collection (List)
- Higher Order Function (.where dan .fold)
- Custom Exception
- Async/Await

# Cara Menjalankan (Di terminal)
dart pub get
dart run bin/main.dart
